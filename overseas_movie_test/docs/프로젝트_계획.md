# 해외 프로젝트 계획

## Project Overview
- Topic: overseas movie box-office analysis
- Analytical subject: distributor-centered interpretation
- Market scope: U.S. theatrical general movies
- Year range: `2016-2025`
- Comparison periods:
  - `2016-2019` pre-COVID
  - `2020-2022` COVID
  - `2023-2025` post-COVID
- Final analysis file:
  - `data/analysis_ready/overseas_us_general_theatrical_2016_2025_analysis_ready.csv`

## Distributor Objectives
1. Identify which genres show stronger average box-office outcomes for distributors.
2. Check which release months and timing windows are more favorable for theatrical distribution.
3. Examine whether rating categories are associated with different box-office outcomes.
4. Compare pre-COVID, COVID, and post-COVID patterns to support distributor planning under changing market conditions.

## Analysis Questions
1. Which genres show relatively stronger box-office performance from a distributor perspective?
2. How do release timing and release month relate to box-office performance?
3. Are there meaningful performance differences across U.S. rating groups?
4. How strongly are theater scale indicators related to box-office outcomes?
5. How did box-office patterns change across pre-COVID, COVID, and post-COVID periods?

## Differences From The Domestic KOBIS Project
- Overseas uses `opening_theaters` and `widest_release_theaters` instead of `screen_count_peak`
- Overseas does not have a reliable `show_count_total` equivalent
- Overseas does not have audience count, so gross is used as the core performance indicator
- Overseas rating analysis depends on U.S.-centered MPAA labels where available
- `brand_name` and `franchise_name` can be used as optional studio-IP context, not as core assignment variables
