# 프로젝트 계획

## 주제
- 미국 극장 개봉 영화 흥행 분석

## 기본 방향
- 미국 데이터를 메인 분석 대상으로 사용한다.
- 한국 데이터는 비교와 해석 보강을 위한 참고 자료로 유지한다.
- 과제 요구사항에 맞게 피처와 종속변수의 관계를 설명할 수 있는 구조로 정리한다.

## 현재 데이터 기준
- 메인 데이터 파일:
  - `미국영화흥행분석/data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터 파일:
  - `미국영화흥행분석/data/참고데이터/한국영화_참고데이터_2016_2025.csv`

## 다음 단계
1. 미국 데이터 기준으로 분석 목적을 한 문장으로 정리한다.
2. 종속변수를 정하고, 설명력 있는 피처 후보를 2개 이상 고른다.
3. 분포도와 상관계수 분석을 수행한다.
4. 선택한 피처와 선택 이유를 보고서 문장으로 정리한다.
