# 과제 요구사항 요약

## 1차 과제
- 데이터 출처를 기록한다.
- 분석에 사용할 수 있는 형태로 데이터를 정리한다.
- 분석 주제와 목적을 명확히 한다.
- EDA 중심으로 데이터 구조와 특징을 설명한다.

## 2차 과제
- 목적에 맞는 피처와 종속변수의 관계를 분석한다.
- 분포도와 상관계수를 기본 분석 방법으로 사용한다.
- 필요하면 데이터를 그룹으로 나누어 비교한다.
- 선택한 피처를 2개 이상 제시하고, 선택 이유를 설명한다.

## 현재 프로젝트 반영 방식
- 메인 데이터: `미국영화흥행분석/data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터: `미국영화흥행분석/data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 메인 주제: 미국 극장 개봉 영화 흥행 성과와 관련된 피처 분석
- 보조 활용: 한국 영화 데이터는 비교와 해석 보강용 참고 자료로 사용
