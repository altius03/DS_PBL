# 과제 요구사항

## 이번 프로젝트에서 맞추려는 과제 방향
- 데이터 출처를 명확히 기록한다.
- 분석 가능한 형태로 데이터를 정리한다.
- 종속변수와 피처의 관계를 분포도와 상관계수로 설명한다.
- 피처를 2개 이상 선택하고, 선택 이유를 정리한다.

## 현재 프로젝트 기준 적용
- 메인 분석 데이터:
  - `data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터:
  - `data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 기본 종속변수 후보:
  - `전세계_흥행수익_USD`
  - `국내_흥행수익_USD`
- 기본 피처 후보:
  - `최대_상영극장수`
  - `오픈_극장수`
  - `장르`
  - `관람등급`
  - `IMDb_평균평점`
  - `TMDB_원어`

## 현재 판단
- 과제 수행에 필요한 데이터는 이미 확보된 상태다.
- 추가 수집보다 현재 데이터로 분석 문장과 시각화를 정리하는 편이 효율적이다.

## 실행하면서 구체화한 방향
- 종속변수는 `전세계_흥행수익_USD`로 고정했다.
- 최종 피처는 `최대_상영극장수`, `오픈_극장수`, `대표장르`, `개봉시기` 4개로 정리했다.
- 수치형 피처는 `분포도 + 산점도 + Pearson 상관계수`로 설명하고, 범주형 피처는 `그룹별 분포 + 중앙값 차이`로 설명하는 방식으로 정리했다.
- `개봉시기`는 미국 시장 특성에 맞춰 `여름 성수기(5~8월)`, `연말 성수기(11~12월)`, `일반기` 3구간으로 재분류했다.

## 문서 및 발표 정리 원칙
- PBL 파일은 중간 과정과 판단 근거를 에세이 스타일로 정리한다.
- PPT는 숫자, 그래프, 선택 이유가 바로 보이도록 기술문서 스타일로 정리한다.
- 발표 문장은 `예측`, `증명`, `영향` 같은 단정적 표현보다 `관련`, `경향`, `의미 있는 변수` 같은 표현을 우선 사용한다.
- 상관관계 결과는 인과관계로 해석하지 않도록 별도 한계 문장을 유지한다.
