# 프로젝트 계획

## 현재 주제
- 미국 극장 개봉 영화 흥행 분석

## 분석 목적
- 흥행 성과와 관련된 피처를 찾고, 그 관계를 과제 형식에 맞게 설명한다.

## 현재 데이터 구성
- 메인 분석 데이터:
  - `data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터:
  - `data/참고데이터/한국영화_참고데이터_2016_2025.csv`

## 우선 분석 후보
1. `최대_상영극장수`와 `전세계_흥행수익_USD`
2. `오픈_극장수`와 `국내_흥행수익_USD`
3. `장르`와 `전세계_흥행수익_USD`
4. `TMDB_미국등급`과 `전세계_흥행수익_USD`
5. `IMDb_평균평점`과 `전세계_흥행수익_USD`

## 진행 순서
1. 종속변수를 정한다.
2. 설명력 있는 피처 2개 이상을 고른다.
3. 분포도와 상관계수를 만든다.
4. 선택 이유와 해석 문장을 정리한다.

## 실행 중 변경된 계획
- 초기 후보에 있던 `오픈_극장수`와 `국내_흥행수익_USD` 조합은, 발표 전체 흐름의 일관성을 위해 `전세계_흥행수익_USD` 기준으로 통일했다.
- `TMDB_미국등급`, `IMDb_평균평점`, `제작비`는 흥미로운 후보였지만, 결측과 coverage 문제를 고려해 이번 2차 과제의 최종 피처에서는 제외했다.
- 결과 설명은 `수치형 2개 + 범주형 2개` 구조로 정리해 슬라이드 흐름을 단순화했다.

## 최종 확정 분석 축
1. `최대_상영극장수`와 `전세계_흥행수익_USD`
2. `오픈_극장수`와 `전세계_흥행수익_USD`
3. `대표장르`와 `전세계_흥행수익_USD`
4. `개봉시기`와 `전세계_흥행수익_USD`

## 최종 산출물 방향
- 분석 표와 그래프는 `outputs/assignment2/` 기준으로 정리한다.
- PPT는 `왜 이 피처를 골랐는가`가 중심이 되도록 수정한다.
- 제출본은 `submissions/assignments/2차과제/` 기준으로 보관한다.
