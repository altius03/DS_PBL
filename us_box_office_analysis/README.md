# 미국영화흥행분석

미국 극장 개봉 영화 데이터를 중심으로 흥행 성과와 관련 피처를 분석하는 프로젝트다.
한국 영화 데이터는 비교와 해석 보강을 위한 참고 자료로 함께 보관한다.

## 현재 기준
- 메인 분석 파일: `data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 파일: `data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 기준 기간: `2016-2025`
- 오늘 기준 진행 과제: `2차 과제`

## 핵심 폴더
- `data/분석데이터/`
  - 분석에 바로 사용하는 데이터
- `data/수집자료/`
  - 수집 과정에서 나온 CSV, 리포트, 인덱스 파일
- `docs/`
  - 과제 요구사항, 비교 메모, 작업 기록, 출처 기록
- `src/`
  - 수집, 보강, 시각화 스크립트
- `outputs/assignment2/`
  - 2차 과제 산출물
- `archive/`
  - 이전 구조와 레거시 산출물 보관

## 2차 과제 산출물
- 시각화: `outputs/assignment2/visuals/`
- 요약표: `outputs/assignment2/tables/`
- 생성 스크립트: `src/build_assignment2_visuals.py`

## 해석상 주의점
- 미국 데이터에는 정확한 관객 수와 총 상영 횟수가 없다.
- `opening_theaters`, `widest_release_theaters`는 KOBIS의 스크린 수와 완전히 같은 개념이 아니다.
- 한국 데이터는 참고 자료이며, 현재 프로젝트의 메인 분석 대상은 미국 데이터다.
