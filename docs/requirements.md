# Requirements

## 1차 과제 요구사항
1. 팀 주제를 선택한다.
2. 데이터 분석을 수행하는 주체와 목적을 4개 이상 제시한다.
3. 분석에 필요한 데이터를 확보하고 출처와 파일을 정리한다.
4. 전체 데이터를 이해하기 위한 기본 분석을 수행한다.
   - 기본 통계량
   - 결측치 비율
   - 변수 분포 확인
   - 그래프 기반 시각화
5. 제출물을 정리한다.
   - PPT
   - 데이터 파일
   - 코드 파일

## 2차 과제 요구사항
원문 요구사항 번호를 유지해 `1, 3, 4` 기준으로 정리한다.

### 1. 피처 선택 분석
- 목적에 맞는 피처(혹은 변환된 애트리뷰트)와 종속변수의 관계를 분석한다.
- 적합한 피처를 2개 이상 선택한다.
- 분석 방법은 분포도, 상관계수를 기본으로 사용한다.
- 필요하면 그룹으로 데이터를 분할해 분석해도 된다.
- 선택한 피처와 선택 이유를 설명해야 한다.
- 추가 데이터가 필요하면 확보해도 된다.
- 필요하면 피처 가공을 해도 된다.
- 진행이 잘 안 되면 목적을 조정해도 된다.

### 3. 학습 내용 정리
- 강의자료와 별도로 스스로 찾고 학습한 방법을 정리한다.
- 노션, 블로그 등에 정리하고 링크를 제출한다.

### 4. 결과 제출
- PBL 파일(서식 2 혹은 3)
- PPT 파일
  - 1차 결과에 2차 진행 내용을 누적해서 제출
- 코드 파일

## 2차 과제 해석
- 먼저 종속변수를 정해야 한다.
- 종속변수와 관계가 있는 피처를 데이터로 설명해야 한다.
- 분포도와 상관계수는 필수 근거로 가져가는 것이 안전하다.
- 범주형 변수는 그룹 분할과 비교 그래프로 보완하는 것이 좋다.
- 최종적으로는 “왜 이 피처를 골랐는가”를 설명할 수 있어야 한다.

## 발표 자료 연결 포인트
- 1차는 데이터 구조를 이해했다는 점을 강조
- 2차는 종속변수 기준으로 피처를 선별했다는 점을 강조
- 단순히 그래프를 나열하지 말고, 선택 근거를 연결해서 설명
