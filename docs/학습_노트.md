# 학습 노트

## 이번 프로젝트에서 확인한 점
- 미국 영화 데이터는 관객 수보다 흥행수익과 극장 수 지표가 더 현실적인 분석 기준이 된다.
- 한국 영화 데이터는 관객 수와 상영 횟수 해석에 강하고, 미국 데이터는 수익과 개봉 규모 해석에 강하다.
- 미국 데이터는 `IMDb`, `TMDB` 보강을 통해 피처 폭을 넓히는 방식이 효율적이었다.
- 파일 구조와 이름은 분석 목적 중심으로 단순하게 두는 편이 훨씬 관리하기 쉽다.

## 실무적으로 중요했던 점
- 원본, 수집 중간 산출물, 최종 분석 데이터를 분리하면 정리가 쉬워진다.
- 중간 보강 산출물은 최종 파일을 만든 뒤 정리하는 편이 좋다.
- 문서와 실제 파일 경로가 어긋나기 시작하면 이후 작업 비용이 크게 늘어난다.

## 현재 기준 메모
- 세부 학습 내용은 `미국영화흥행분석/docs/학습_노트.md`에 계속 누적한다.
