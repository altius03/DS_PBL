# 작업 기록

- KOBIS 기반 한국 영화 데이터 수집과 1차 정리를 진행했다.
- 한국 영화 분석용 CSV와 미매칭 확인용 CSV를 분리해 관리했다.

- Box Office Mojo를 중심으로 미국 개봉 일정과 흥행 데이터를 정리했다.
- 미국 일반 극장 영화 분석용 데이터셋을 구축했다.

## 2026-04-17
- IMDb bulk로 평점, 투표수, 기본 메타데이터를 보강했다.
- TMDB로 미국 등급, 제작사, 제작국가, 원어를 보강했다.
- 프로젝트 구조를 미국 중심으로 재편했다.
- 한국 데이터는 하나의 참고용 CSV로 묶어 메인 프로젝트 내부로 이동했다.
- 메인 프로젝트 이름을 `미국영화흥행분석`으로 정리했다.
