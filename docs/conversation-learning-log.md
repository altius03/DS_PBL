## 2026-04-14 / KOBIS 데이터 이해
### 학습 내용
- KOBIS 연도별 박스오피스 다운로드 파일은 `.xls` 확장자라도 실제로는 HTML 테이블 형태일 수 있다.
- 연도별 박스오피스는 해당 연도 개봉작만 의미하지 않고, 그 해에 매출이 발생한 영화 전체가 포함될 수 있다.
- 따라서 연도별 박스오피스 원본을 그대로 분석하면 재개봉작과 구작이 함께 섞일 수 있다.

### 이번 프로젝트에 적용한 판단
- 원본 박스오피스 데이터는 기준 테이블로 사용한다.
- 메타정보 파일을 붙여 장르, 관람등급, 영화코드, 감독, 제작사를 보강한다.
- 최종 분석용 데이터셋은 개봉일 기준 `20160101 ~ 20251231` 범위로 다시 필터링한다.

### 확인한 처리 방식
- `영화명 + 개봉일` 기준으로 우선 매칭
- 분석 범위 내 매칭 성공: `5171`
- 분석 범위 내 미매칭: `5`

### 메모
- 미매칭 항목은 공연 실황·팬 콘서트 성격일 가능성이 높다.
- 장르는 다중 값(`|`)이 포함될 수 있으므로 메인 장르 처리 기준을 정해야 한다.
