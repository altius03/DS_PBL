# Project Overview

## 프로젝트 스냅샷
- 주제: 영화
- 분석 주체: 배급사
- 데이터 범위: 한국 일반영화
- 분석 기간: 2016~2025
- 팀원 수: 2명
- 현재 단계: 1차 과제 기획 및 데이터 수집 시작 단계
- 1차 마감: 2026-04-14
- 이후 계획: 같은 주제로 2차 과제까지 확장
- 권장 범위: 국내 영화 / 국내 박스오피스 중심

## 핵심 전략
- 1차 과제는 데이터 이해(EDA) 중심으로 안정적으로 마무리한다.
- 2차 과제는 종속변수와 피처의 관계를 분석해 적합한 피처를 2개 이상 선택하는 방향으로 확장한다.
- 2차 분석 근거는 분포도와 상관계수를 중심으로 잡는다.
- 필요하면 그룹 분할, 추가 데이터 확보, 피처 가공을 사용한다.

## 결과물 퀄리티 목표
- 현재 목표는 `완성도 강화형`이다.
- 의미:
  - 제출만 가능한 최소 수준이 아니라, 발표에서 설득력 있게 보일 정도의 완성도를 목표로 한다.
  - 다만 1차 과제에서는 복잡한 예측 모델보다 EDA 품질과 해석의 선명함을 우선한다.
- 1차에서 완성도 강화형으로 본다는 뜻:
  - 데이터 출처와 범위가 명확하게 정리되어 있을 것
  - 전처리 기준과 결측치 처리 방침이 설명 가능할 것
  - 그래프가 5~7개 수준으로 충분히 준비되어 있을 것
  - 각 그래프마다 한 줄 이상의 해석 문장이 붙을 것
  - 2차 과제로 자연스럽게 이어질 변수 후보가 보일 것
- 피해야 할 것:
  - 범위를 과도하게 넓혀 정작 설명이 흐려지는 결과물
  - 모델링 욕심 때문에 1차 EDA 품질이 떨어지는 경우

## 권장 제목
- 1차 과제: 국내 영화 흥행 데이터의 구조와 특성 분석
- 2차 과제: 국내 영화 흥행 성과와 주요 피처의 관계 분석 및 핵심 피처 선택

## 추천 데이터 소스
- KOBIS / KOFIC 계열 데이터
  - 박스오피스, 개봉일, 배급사, 관객 수, 매출액, 스크린 수, 상영 횟수 확보에 적합
- 공공데이터포털 / 영화진흥위원회 제공 데이터
  - 국내 영화 메타데이터 보완에 적합
- 보조 후보
  - TMDb / IMDb / Kaggle
  - 해외 정보나 보조 메타데이터가 꼭 필요할 때만 제한적으로 사용

## 분석 주체/목적 후보
- 배급사: 어떤 장르, 개봉 시기, 관람등급의 영화가 높은 관객 수와 매출을 보이는가
- 제작사: 흥행 가능성이 높은 영화의 특징은 무엇인가
- 극장 체인: 스크린 수와 상영 횟수는 흥행 성과와 어떤 관계가 있는가
- 투자사: 흥행 상위 영화와 일반 영화의 차이는 무엇인가

## 현재 확정한 분석 주체
- 배급사
- 해석:
  - 1차 EDA에서 장르, 개봉 시기, 관람등급, 스크린 수, 상영 횟수, 흥행 성과를 자연스럽게 연결하기 좋다.
  - 2차 과제에서 종속변수와 피처 관계를 분석하는 방향으로 확장하기 유리하다.

## 현재 확정한 목적 5개
1. 배급사 입장에서 어떤 장르의 영화가 더 높은 흥행 성과를 보이는가?
2. 배급사 입장에서 어떤 개봉 시기(월·분기·계절)가 흥행에 유리한가?
3. 배급사 입장에서 관람등급에 따라 흥행 성과 차이가 존재하는가?
4. 배급사 입장에서 스크린 수와 상영 횟수는 흥행 성과와 어떤 관계가 있는가?
5. 배급사 입장에서 코로나 이전·코로나기·최근 구간에 흥행 패턴은 어떻게 달라졌는가?

## 목적 활용 구조
- 1차 과제에서는 위 5개 목적을 모두 활용한다.
- 목적들은 서로 분리된 질문이 아니라, 배급사 입장에서 흥행 성과와 관련된 요인을 이해하기 위한 연결된 질문으로 본다.
- 시기별 변화 목적은 나머지 목적들을 해석할 때 함께 쓰는 비교 축 역할도 한다.
- 2차 과제에서는 필요하면 이 중 하나를 중심 목적로 선택해 더 깊게 확장한다.

## 현재 확정한 데이터 범위
- 한국 영화만 포함한다.
- 일반영화 중심으로 분석한다.
- 독립영화·예술영화는 우선 제외한다.
- 외국 영화는 우선 제외한다.

## 현재 확정한 기간 범위
- 전체 분석 기간은 2016~2025의 10년으로 설정한다.
- 시기 구간은 아래처럼 나눈다.
  - 2016~2019: 코로나 이전
  - 2020~2022: 코로나기
  - 2023~2025: 최근 구간

## 기간 범위 선택 이유
- 10년 단위는 한 학기 프로젝트로서 충분한 규모를 갖추면서도 관리 가능한 범위다.
- 최신 완결 연도인 2025까지 포함해 현재 시점 기준의 최신성을 확보할 수 있다.
- 시기별 비교를 통해 최근 한국 영화 산업의 변화와 침체 흐름을 설명하기 좋다.
- 2016~2025 범위는 연도별 추이와 구간별 비교를 모두 수행하기에 적절하다.

## 현재 확정한 데이터 소스 전략
- 1차와 초기 분석은 KOBIS만 사용한다.
- 보조 데이터 소스는 당장 붙이지 않는다.
- 실제 수집과 분석 후 부족한 점이 확인되면 그때 보조 소스를 재검토한다.

## 데이터 소스 전략 선택 이유
- KOBIS만으로도 현재 목적 5개를 설명하는 데 필요한 핵심 변수 확보 가능성이 높다.
- 데이터 출처를 하나로 두면 수집과 전처리, 병합 관리가 단순해진다.
- 1차 과제에서 데이터 병합 난도로 시간을 소모하지 않고 EDA 품질에 집중할 수 있다.
- 추가 소스는 실제 부족함이 확인된 뒤 붙이는 편이 더 효율적이다.

## 데이터 범위 선택 이유
- 독립영화·예술영화까지 포함하면 개봉 규모, 스크린 수, 상영 횟수 구조가 크게 달라져 비교 해석이 흔들릴 수 있다.
- 독립영화·예술영화는 데이터 정리와 품질 관리 난도가 상대적으로 높다.
- 배급사 관점이라 하더라도 한국 영화와 외국 영화를 함께 넣으면 시장 구조와 흥행 패턴 차이로 인해 과제 규모가 과도하게 커질 수 있다.
- 한국 일반영화로 범위를 좁히면 데이터 확보, 시각화, 발표 메시지, 2차 과제 확장성을 모두 안정적으로 가져갈 수 있다.

## 현재 제안하는 최소 컬럼 목록
목적 5개를 1차 과제에서 모두 다루기 위해, 수집 전 기준으로 아래 컬럼을 우선 확보한다.

### 필수 컬럼
- `movie_id`
  - 영화별 식별값. 데이터 병합과 중복 제거에 필요
- `movie_name`
  - 결과물 설명과 발표 표기에 필요
- `open_date`
  - 개봉 시기 분석의 기준값
- `genre`
  - 장르별 흥행 성과 비교에 필요
- `rating`
  - 관람등급별 성과 비교에 필요
- `sales_acc`
  - 누적매출액. 흥행 성과 지표 후보
- `audience_acc`
  - 누적관객수. 흥행 성과 지표 후보
- `screen_count`
  - 스크린 수와 흥행 성과 관계 분석에 필요
- `show_count`
  - 상영 횟수와 흥행 성과 관계 분석에 필요
- `nation`
  - 한국 영화만 필터링하기 위해 필요
- `movie_category`
  - 일반영화 여부를 구분하기 위한 값

### 보조 컬럼
- `distributor`
  - 배급사 관련 설명을 보강할 때 유용
- `production_company`
  - 제작사 정보가 필요할 경우 보조적으로 사용
- `director`
  - 영화 설명용 메타데이터
- `runtime`
  - 보조 분석이나 2차 확장 시 활용 가능
- `year`
  - 연도별 집계에 바로 활용하기 편함
- `month`
  - 월별 비교에 바로 활용하기 편함
