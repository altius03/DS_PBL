# Analysis Plan

## 1차 과제 목표
- 데이터 구조와 품질을 이해한다.
- 2차 과제에서 바로 사용할 수 있는 핵심 변수들을 확인한다.
- 발표에 넣을 수 있는 시각화 3~5개를 만든다.

## 1차 EDA 체크리스트
- 데이터 shape 확인
- 컬럼 타입 정리
- 결측치 개수와 비율 확인
- 중복 여부 확인
- 주요 수치형 변수 기술통계 확인
- 범주형 변수 빈도 확인
- 이상치와 치우친 분포 확인

## 1차 권장 시각화
- 관객 수 분포 히스토그램
- 매출액 분포 히스토그램
- 장르별 영화 수 막대그래프
- 장르별 평균 또는 중앙 관객 수 비교
- 개봉 월별 영화 수 또는 평균 성과 그래프
- 관람등급별 성과 비교 그래프
- 스크린 수 vs 관객 수 산점도
- 상영 횟수 vs 매출액 산점도

## 2차 과제 기본 방향
- 종속변수 후보를 먼저 정한다.
- 후보 피처를 수치형과 범주형으로 나눈다.
- 피처별로 종속변수와의 관계를 시각화하고 수치로 확인한다.
- 근거가 분명한 피처 2개 이상을 최종 선택한다.

## 2차 종속변수 후보
- `audience_acc`
- `sales_acc`
- `log_audience_acc`
- `log_sales_acc`

## 2차 피처 후보
- 수치형
  - `screen_count`
  - `show_count`
  - `runtime`
  - `month`
- 범주형
  - `genre`
  - `rating`
  - `season`
  - `distributor`
- 파생 피처
  - `screen_efficiency`
  - `show_efficiency`
  - `genre_main`

## 2차 분석 방법
### 분포도
- 종속변수 분포 확인
- 로그 변환 필요성 확인
- 피처별 분포와 이상치 확인

### 상관계수
- 수치형 피처와 종속변수의 상관계수 계산
- 절대값 크기만 보지 말고 해석 가능성도 함께 본다.

### 그룹 분할
- 장르별
- 관람등급별
- 개봉 시기별
- 필요하면 흥행 상위/하위 그룹 분할

## 피처 선택 기준
- 결측치가 지나치게 많지 않을 것
- 종속변수와 관계가 설명 가능할 것
- 그래프와 수치가 같은 방향을 가리킬 것
- 발표에서 직관적으로 설명할 수 있을 것
- 추가 데이터 없이도 재현 가능할 것

## 2차 제출 시 반드시 설명할 것
- 어떤 종속변수를 선택했는가
- 어떤 피처 2개 이상을 선택했는가
- 왜 그 피처를 선택했는가
- 어떤 그래프와 상관계수를 근거로 삼았는가
- 추가 데이터나 피처 가공을 썼다면 왜 필요했는가

## 오늘 기준 최소 액션 플랜
1. 데이터 범위를 국내 영화로 확정
2. 원본 데이터 1~2개 확보
3. 전처리 및 EDA 수행
4. 그래프 3~5개 생성
5. PPT 초안 반영
