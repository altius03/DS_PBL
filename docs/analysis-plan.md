# Analysis Plan

## 1차 과제 목표
- 데이터 구조와 특성을 EDA로 설명한다.
- 발표에 바로 넣을 수 있는 그래프와 해석 문장을 확보한다.
- 2차 과제로 확장 가능한 피처 후보를 미리 확인한다.

## 분석 기준 데이터
### 최종 분석용
- `data/processed/kobis_master_2016_2025.csv`

### 보조 확인용
- `data/processed/kobis_processing_summary_2016_2025.json`
- `data/processed/kobis_unmatched_movies_2016_2025.csv`
- `data/processed/kobis_master_boxoffice_scope_2016_2025.csv`

## 처리 기준
- 연도별 박스오피스 원본 10개 파일을 통합한다.
- 메타정보 파일을 별도로 정리한다.
- `영화명 + 개봉일` 기준으로 우선 매칭한다.
- 최종 분석 범위는 개봉일 기준 `20160101 ~ 20251231`만 남긴다.

## 현재 데이터 상태
- 분석 범위 영화 수: `5176`
- 메타데이터 매칭 성공: `5171`
- 미매칭: `5`

## 1차 분석 질문
1. 장르와 흥행 성과의 관계는 어떠한가
2. 개봉 시기와 흥행 성과의 관계는 어떠한가
3. 관람등급에 따라 흥행 성과 차이가 존재하는가
4. 스크린 수와 상영 횟수는 흥행 성과와 어떤 관계를 보이는가
5. 시기 구간에 따라 흥행 패턴이 어떻게 달라지는가

## 전처리 체크리스트
- 개봉일 형식 확인
- 장르 다중 값 처리 방식 결정
- 관람등급 범주 정리
- 배급사명 표기 차이 확인
- 결측치 확인
- 미매칭 5개 수동 보완 여부 결정

## 권장 파생변수
- `open_year`
- `open_month`
- `quarter`
- `season`
- `covid_period`
- `genre_main`

## 1차 EDA 항목
- 데이터 크기와 컬럼 확인
- 결측치 현황
- 관객수·매출액 분포
- 장르별 영화 수와 흥행 성과 비교
- 월별 또는 분기별 흥행 성과 비교
- 관람등급별 흥행 성과 비교
- 스크린 수 vs 관객수 산점도
- 상영 횟수 vs 매출액 산점도
- 시기 구간별 비교

## 발표용 그래프 후보

- 매출액 분포 히스토그램

- 장르별 평균 또는 중앙 관객수 비교
- 월별 개봉 편수 또는 월별 평균 성과 그래프
- 관람등급별 성과 비교 그래프
- 스크린 수와 관객수 산점도
- 상영 횟수와 매출액 산점도
- 시기 구간별 비교 그래프
