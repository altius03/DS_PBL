# Project Plan

## 프로젝트 스냅샷
- 주제: 영화
- 분석 주체: 배급사
- 분석 범위: 한국 일반영화
- 분석 기간: 2016~2025
- 시기 구간: 2016~2019 / 2020~2022 / 2023~2025
- 데이터 소스: KOBIS 우선 사용
- 현재 단계: 분석용 CSV 정리 완료, 1차 시각화 초안 생성 완료

## 총괄 목적
- 배급사 관점에서 한국 일반영화의 흥행 성과와 관련된 주요 요인을 파악하고, 향후 배급 전략 수립에 활용할 기초 자료를 마련하고자 한다.

## 세부 분석 질문
1. 배급사 입장에서 어떤 장르의 영화가 더 높은 흥행 성과를 보이는가
2. 배급사 입장에서 어떤 개봉 시기가 흥행에 유리한가
3. 배급사 입장에서 관람등급에 따라 흥행 성과 차이가 존재하는가
4. 배급사 입장에서 스크린 수와 상영 횟수는 흥행 성과와 어떤 관계가 있는가
5. 배급사 입장에서 코로나 이전·코로나기·최근 구간에 흥행 패턴은 어떻게 달라졌는가

## 1차 과제 방향
- 1차는 EDA 중심으로 진행한다.
- 기본 통계, 결측치, 변수 분포, 범주형 비교, 시각화 해석에 집중한다.
- 2차 과제에 필요한 피처 후보는 메모하되, 관계 분석 자체는 2차에서 확장한다.

- 메타정보 CSV 1개

### `data/raw/`
- 실제 작업용 CSV 정리본
- 박스오피스 CSV 10개
- 메타정보 CSV 1개

### `data/analysis_ready/`
- 최종 분석용 통합 CSV
- 미매칭 확인용 CSV

## 해석 시 주의점
- 연도별 박스오피스는 해당 연도 개봉작만 의미하지 않는다.
- 원본 `.xls` 파일은 실제로 HTML 기반 다운로드 형식이다.
- 장르는 다중 값(`|`)을 포함할 수 있다.
- 미매칭 5개는 공연 실황·팬 콘서트 계열일 가능성이 높다.
