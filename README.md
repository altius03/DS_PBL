# DS_PBL

현재 저장소의 메인 작업공간은 `us_box_office_analysis/`이다.

## 현재 기준
- 메인 분석 주제: 미국 극장 개봉 영화 흥행 분석
- 메인 데이터 파일: `us_box_office_analysis/data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터 파일: `us_box_office_analysis/data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 기준 기간: `2016-2025`
- 오늘 기준 진행 과제: `2차 과제`

## 주요 작업 위치
- `us_box_office_analysis/`
  - 실제 과제 진행과 데이터 정리가 이루어지는 메인 폴더
- `us_box_office_analysis/src/`
  - 데이터 수집, 보강, 시각화 스크립트
- `us_box_office_analysis/outputs/assignment2/`
  - 2차 과제 산출물
- `archive/`
  - 루트 레거시 폴더와 이전 구조 백업

## 문서 안내
- `us_box_office_analysis/README.md`
  - 메인 프로젝트 개요
- `us_box_office_analysis/docs/`
  - 과제 요구사항, 비교 메모, 작업 기록, 출처 기록

## 메모
- 루트 레벨의 예전 작업 폴더는 `archive/`로 옮겨 관리한다.
- 미국 데이터가 메인이고, 한국 데이터는 비교와 참고 목적의 보조 자료로 정리되어 있다.
