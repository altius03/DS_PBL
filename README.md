# DS_PBL

현재 저장소의 메인 작업공간은 `미국영화흥행분석/`이다.

## 현재 기준
- 메인 분석 주제: 미국 극장 개봉 영화 흥행 분석
- 메인 데이터 파일: `미국영화흥행분석/data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터 파일: `미국영화흥행분석/data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 기준 기간: `2016-2025`
- 분석 방식: 분포도, 그룹 비교, 상관계수 중심

## 폴더 안내
- `미국영화흥행분석/`
  - 실제 과제 진행과 데이터 정리가 이루어지는 메인 폴더
- `docs/`
  - 루트 기준 간단한 요약 문서
- `presentation/`
  - 발표 자료 작업 폴더
- `src/`
  - 루트 공용 스크립트가 필요한 경우 사용하는 폴더

## 문서 안내
- `미국영화흥행분석/README.md`
  - 메인 프로젝트 개요
- `미국영화흥행분석/docs/`
  - 과제 요구사항, 비교 메모, 작업 기록, 출처 기록
- `docs/`
  - 루트 수준 요약 문서

## 메모
- 한국 데이터는 더 이상 루트 `data/`에 흩어져 있지 않다.
- 미국 데이터가 메인이고, 한국 데이터는 비교와 참고 목적의 보조 자료로 정리되어 있다.
