# 출처 기록

## 미국 데이터 기준 출처
### Box Office Mojo
- 미국 극장 개봉 일정 수집의 기준 출처
- 월별 일정 페이지 패턴:
  - `https://www.boxofficemojo.com/calendar/YYYY-MM-01/`
- 세부 페이지 패턴:
  - `https://www.boxofficemojo.com/releasegroup/{id}/`
  - `https://www.boxofficemojo.com/release/{id}/`
  - `https://www.boxofficemojo.com/title/{imdb_title_id}/`
  - `https://www.boxofficemojo.com/title/{imdb_title_id}/credits/`
- 수집 범위:
  - `2016-2025`
  - 미국 극장 개봉작
- 수집 규모:
  - 월별 일정 취합 후 중복 제거 backbone `7605행`
  - 일반 영화 필터 적용 후 최종 분석 대상 `7368행`
- 주요 수집 항목:
  - 영화명
  - 개봉일
  - 배급사
  - 개봉 규모
  - 장르
  - 관람등급
  - 전 세계 흥행수익
  - 미국 내 흥행수익
  - 해외 흥행수익
  - 오프닝 수익
  - 개봉 극장 수
  - 최대 상영 극장 수
  - 상영시간
- 제목 페이지 추가 수집 항목:
  - 제작비
  - 브랜드명
  - 프랜차이즈명
  - 제목 페이지 보조 수익값
  - 감독
  - 각본
  - 제작자

### IMDb Bulk Data
- 보강 출처
- 다운로드 기준:
  - `datasets.imdbws.com`
  - `title.ratings.tsv.gz`
  - `title.basics.tsv.gz`
- 매칭 기준:
  - 기존 데이터의 `imdb_title_id`
- 주요 보강 항목:
  - IMDb 평점
  - IMDb 투표수
  - 작품 유형
  - 기본 제목
  - 원제목
  - 성인물 여부
  - 시작연도
  - 상영시간
  - IMDb 장르

### TMDB API
- 보강 출처
- API 기준 주소:
  - `https://api.themoviedb.org/3`
- 주요 조회 흐름:
  - `find/{imdb_id}?external_source=imdb_id`
  - `movie/{tmdb_id}?append_to_response=release_dates`
- 매칭 기준:
  - `imdb_title_id`
- 주요 보강 항목:
  - TMDB 영화 ID
  - 미국 관람등급
  - 제작사
  - 제작국가
  - 원어

### 보조 시도 출처
- Wikidata Query Service
- 엔드포인트:
  - `https://query.wikidata.org/sparql`
- 시도 항목:
  - 제작사
  - 제작국가
  - 원어
- 메모:
  - IMDb ID 기준으로 매칭은 되었지만, 해당 항목들의 일관성이 낮아 최종 보강의 주력 출처로는 사용하지 않았다.

## 수집 및 정제 메모
- 월별 일정 페이지는 여러 달에 걸쳐 같은 영화가 중복 노출되므로 URL 기준 중복 제거를 수행했다.
- 일정 페이지에는 다음 해 개봉작이 섞일 수 있어 최종 연도 범위를 다시 잘랐다.
- 일반 영화 필터링 과정에서 재개봉, 공연 실황, 이벤트 시네마, 연속 상영 프로그램, 단편성 항목을 제외했다.
- 미국 데이터에는 KOBIS식 `관객 수`, `총 상영 횟수`, `정확한 스크린 수`가 안정적으로 포함되지 않는다.
