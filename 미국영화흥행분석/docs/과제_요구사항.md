# 과제 요구사항

## 이번 프로젝트에서 맞추려는 과제 방향
- 데이터 출처를 명확히 기록한다.
- 분석 가능한 형태로 데이터를 정리한다.
- 종속변수와 피처의 관계를 분포도와 상관계수로 설명한다.
- 피처를 2개 이상 선택하고, 선택 이유를 정리한다.

## 현재 프로젝트 기준 적용
- 메인 분석 데이터:
  - `data/분석데이터/미국영화_분석데이터_2016_2025.csv`
- 참고 데이터:
  - `data/참고데이터/한국영화_참고데이터_2016_2025.csv`
- 기본 종속변수 후보:
  - `전세계_흥행수익_USD`
  - `국내_흥행수익_USD`
- 기본 피처 후보:
  - `최대_상영극장수`
  - `오픈_극장수`
  - `장르`
  - `관람등급`
  - `IMDb_평균평점`
  - `TMDB_원어`

## 현재 판단
- 과제 수행에 필요한 데이터는 이미 확보된 상태다.
- 추가 수집보다 현재 데이터로 분석 문장과 시각화를 정리하는 편이 효율적이다.
